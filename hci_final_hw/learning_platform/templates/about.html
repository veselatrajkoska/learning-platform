About us

This system is a platform is intended for anyone interested in learning about machine learning. The platform is organized in a way that even someone with no prior knowledge of the topic could take up the machine learning course, since it also includes courses for certain skills necessary to master it – programming in Python, data analysis and probability.

The platform offers a simple interface for navigating through the screens. A user can scroll through the courses and view their contents and start a course or multiple courses at the same time. Each course offers a variety of lessons.

After each module, the user needs to complete a short test to be able to continue onto the next. The tasks comprise of a variety of tasks – short theoretical questions, problem solving tasks and longer programming problems.
